What is PNG and why use it?

PNG (Portable Network Graphics) is a raster graphics file format that supports lossless data compression.
The reason to use PNG in this instance is that it provides high-quality images with support for transparency. This makes it ideal for web graphics, logos, and images with text. The downside of PNG is that it typically results in larger file sizes compared to JPG, especially for photographs.
