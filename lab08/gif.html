What is GIF and why use it?

GIF (Graphics Interchange Format) is a bitmap image format that supports both static and animated images.
The reason to use GIF is that it allows for simple animations and supports transparency, making it ideal for graphics with limited colors. The downside of GIF is that it uses a limited color palette (256 colors), which can result in lower image quality for photographs and complex images.
