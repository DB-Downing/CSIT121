What is JPG and why use it?

JPG (Joint Photographic Experts Group) is a commonly used method of lossy compression for digital images.
The reason to use JPG is that it captures a lot of image detail in a relatively small file size, making it ideal for photographs and realistic images on the web. The downside of JPG is that it uses lossy compression, which means that some image quality is sacrificed to reduce file size, and it can not be formatted to be transparent.
